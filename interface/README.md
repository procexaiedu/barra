# Interface

Frontend Next.js da Elite Baby.

## Estrutura

- `src/app/(auth)/` concentra rotas de autenticacao.
- `src/app/(interface)/` concentra a interface operacional.
- `src/components/ui/` guarda componentes shadcn/ui.
- `src/lib/` guarda clientes e utilitarios de integracao.
- `src/tipos/` recebe tipos gerados a partir do OpenAPI do backend.

## Comandos

```powershell
pnpm install
pnpm dev
pnpm lint
pnpm build
```
