# Elite Baby

Monorepo do MVP Elite Baby.

## Estrutura

- `api/`: backend Python com FastAPI, LangGraph e ARQ.
- `interface/`: frontend Next.js.
- `infra/`: deploy, compose, SQL e runbooks operacionais.
- `docs/`: documentacao de produto, arquitetura e historico.
- `scripts/`: utilitarios de desenvolvimento.

Leia `docs/estrutura-codebase.md` para a convencao completa da arvore.
